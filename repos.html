Enron Data Processing

A collection of AWK scripts for parsing the Enron email corpus (1.3 GB of unstructured text). This project harnesses the power of regex and associative arrays to efficiently process and clean over 600,000 emails for statistical analysis.

Blackjack

A multi-call card/blackjack program written in C. This CLI program utilizes a custom-made randomness library based on the Xorshift algorithm linked at compile time to perform various functions on a virtual deck.
